## Sync

This repository is automatically updated by [leetcode-syncer](https://github.com/Kartikeyjindal/leetcode-syncer) — a tool that:

1. Fetches accepted submissions from LeetCode
2. Checks if the solution already exists in this repo
3. Pushes only new solutions to GitHub
